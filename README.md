# ViewPager Test

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![ViewPager2](https://img.shields.io/badge/Component-ViewPager2-orange.svg)](https://developer.android.com/jetpack/androidx/releases/viewpager2)

Тестовое приложение для изучения ViewPager2 в Android. Демонстрирует создание горизонтальных и вертикальных page-интерфейсов с кастомными анимациями.

## Функциональность

- Горизонтальный ViewPager2 с флагами стран
- Вертикальный ViewPager2 с цветовыми страницами
- Параллакс-эффект для текста при прокрутке
- Splash Screen при запуске приложения
- Кастомные адаптеры для разных типов данных

## Технологии

- Kotlin
- ViewPager2
- ViewBinding
- Splash Screen API
- Кастомные PageTransformer
